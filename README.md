# Описание проекта
Это проект, в котором я изучал стандартные контейнеры из **STL**. 
Здесь я написал свой **односвязный список**, в нем я реализовал многие методы стандартных контейнеров и добавил немного своих.
### Функционал:
1. Добавление элемента
	* В начало списка
	* В конец списка
	* Вставка на указанную позицию
2. Удаление элемента
	* По значению
		* Первый
		* Все
	* По позиции
		* Удаление одного элемента
		* Удаление элементов в диапазоне индексов
3. Поиск позиции элемента
	* Первого элемента
	* Всех элементов
4. Изменение элемента
	* По позиции через оператор [ ]
	* По значению
		* Первое вхождение
		* Всех элементов
5. Обмен двух элементов местами
6. Вычисление размера списка
7. Вывод списка на консоль
8. Очистка списка
9. Деструктор, который позволяет избежать утечек памяти

Взаимодействие с функционалом списка я сделал через **меню**.
