Это проект, в котором я изучал стандартные контейнеры из STL. 
Здесь я реализую свой односвязный список, в нем я реализовал многие методы стандартных контейнеров и немного добавил своих:
1) Добавление элемента
	a) В начало списка
	b) В конец списка
	c) Вставка на указанную позицию
2) Удаление элемента
	a) По значению
		I) Первый
		II) Все
	b) По позиции
		I) Удаление одного элемента
		II) Удаление элементов в диапазоне индексов
3) Поиск позиции элемента
	a) Первого элемента
	b) Всех элементов
4) Изменение элемента
	a) По позиции через оператор []
	b) По значению
		I) Первое вхождение
		II) Всех элементов
5) Обмен двух элементов местами
6) Вычисление размера списка
7) Вывод списка на консоль
8) Очистка списка
9) Деструктор, который позволяет избежать утечек памяти

Взаимодействие со всем функционалом я сделал через меню.
